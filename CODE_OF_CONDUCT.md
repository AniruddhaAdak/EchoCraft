# Code of Conduct

## Our standard

EchoCraft should remain respectful, practical, and safe for contributors and users.

## Expected behavior

- Be respectful in reviews and discussions.
- Focus criticism on code and product decisions, not people.
- Share context clearly when reporting bugs or proposing changes.
- Help keep the repository safe by avoiding leaked secrets or unsafe guidance.

## Unacceptable behavior

- Harassment, insults, or intimidation.
- Discriminatory or exclusionary language.
- Repeated bad-faith disruption of issues or pull requests.
- Sharing private credentials, tokens, or personal data.

## Reporting

If you encounter unacceptable behavior, open a private support request with the maintainer and include links or screenshots if relevant.
