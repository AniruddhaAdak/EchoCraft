# Content Workflow

1. Upload or record clean audio.
2. Generate a transcript and review the raw output.
3. Translate the transcript if you need a different audience language.
4. Generate a blog draft for longer-form content.
5. Generate a social post for short-form promotion.
6. Open the post workspace to refine, copy, or download the final version.

The best results come from validating the transcript before generating downstream content.
